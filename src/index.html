        Profiles

        A profile is a separate workspace: its own projects, commands, terminal settings, agents, canvas files and screenshots. Switching profiles restarts the app.

        Existing profiles stay in the old folder; this only changes where OctiqFlow looks for them. The app restarts after you pick.

        Terminal

        Choose the font for every terminal in the app. Changes apply right away to open terminals.

        Bundled fonts ship regular and bold faces, so Light/Medium render as Regular and Semi-bold as Bold. System fonts use whichever weights this Mac has installed.

        Agent session resume & alerts

        Let a terminal that runs Claude or Codex re-open its session after OctiqFlow restarts, and flag the tab when the agent is waiting for you. This installs a small hook for each agent that records the tab's session id (so a restored tab runs claude --resume or codex resume on its own) and raises an attention alert when the agent needs your input — so you notice even when you are working in another project. Set it up once.

        Canvas

        The canvas shows a live HTML or Markdown document in a pane beside the terminal. Turn it on with the canvas button in the project's right panel. To let Claude draw and update the canvas during a chat, install the canvas skill — it teaches Claude to write documents into each project's canvas folder ($OCTIQ_CANVAS_DIR). This writes one file to ~/.claude/skills/octiq-canvas/ and changes nothing else. Run it once.

        Codex can drive the canvas too — the channel is just files. This adds a short canvas guide to ~/.codex/AGENTS.md (in a marked block, keeping the rest of the file), so Codex knows to write documents into $OCTIQ_CANVAS_DIR.

        Screenshot vault

        Capture the window you are testing with a global hotkey, even when OctiqFlow is in the background. Each shot goes into a temporary vault. Open the vault from the camera button in the top bar, then paste a screenshot's file path into a terminal so Claude or Codex can read it. You can crop a shot before pasting.

        The hotkey is OFF until you turn it on here. On macOS it needs Input Monitoring permission (for the key listener) and Screen Recording permission (to grab the window). macOS asks for each the first time; grant them in System Settings, then restart OctiqFlow once if the capture comes out blank. On Windows no extra permission is needed.

        —

        Click Record, then press your key combination — at least two keys held together (for example both Command keys, or Cmd + Shift + 2). Release the keys to save. Left and right modifier keys are told apart, so "both Command keys" works.
